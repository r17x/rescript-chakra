# ⚡️ ReScript-Chakra Documentation

## Development

Run ReScript and Next in dev mode:

```console
npm run dev
# OR
yarn run dev
```

## Useful commands

Run ReScript compiler to produce JavaScript code output:

```console
npm run dev:compiler
# OR
yarn run dev:compiler
```
